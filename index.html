Paragons of the Prism

Paragons of the Prism is a 3D local multiplayer sports battle arena. Using your shield and abilities to deflect projectiles, players battle each other in 1v1 or 2v2. This project was done as VFS final project with a team of 5.

I was one of the two programmers in the team, I worked on the basic gameplay of the game, player movement, shield rotation, projectile physics, etc. And on the UI by giving functionality to the menus. I also integrated analytics to the game by sending the game data to a web server and creating a web client that showed the data in graphs and maps.
