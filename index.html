Paragons of the Prism

Previous Next

Paragons of the Prism is a 3D local multiplayer sports battle arena. Using your shield and abilities to deflect projectiles, players battle each other in 1v1 or 2v2. This project was done as VFS final project with a team of 5.

I was one of the two programmers in the team, I worked on the basic gameplay of the game, player movement, shield rotation, projectile physics, etc. And on the UI by giving functionality to the menus. I also integrated analytics to the game by sending the game data to a web server and creating a web client that showed the data in graphs and maps.

School Party

School Party is a single player side scroller educational game made in videogame development class in ITESM with a team of 4. In the game the player controls Daniel or Susy to solve problems at school. The problems where minigames the player had to solve, for example: finding the correct answer to a question in a maze, sorting cards in order, defeating an alien ship and playing memory with math equations.

The goal of the project was to give this game to a teacher so he could change the questions to the minigames, and he could use it with his students. For this we used Parse(unfortunately Parse closed down so the app doesnt work) so the teachers could modify the content in the game by modifing the cloud.

In this project I made the maze and the timeline minigame. They feature a maze generator, kiwi.js structure, a tree data structure and sorting algorithms.

Minesweeper

An interactive web app made for a Vancouver Film School class. This was a solo project. The app is the classic minesweeper found in older Windows OS. The objective of the game is to flag all the bombs hidden in the field, to find the hidden bombs you have to use the clues in the adjacent cells. If a cell is not a bomb it will display the number of adjacent bombs when revealed. To reaveal a cell left-click, right-click to flag cells.
